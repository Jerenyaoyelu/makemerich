---

## 2. 自动化流程

1. 从实时行情抓取基础字段（代码、名称、涨跌幅、换手率、成交额、量比、振幅、行业）
2. 清洗数据（过滤 ST、成交额为 0）
3. 按行业聚合，计算行业联动特征
4. 计算 4 个核心评分因子（均归一化到 0-100）
5. 打分排序，输出候选股

---

## 3. 字段说明

## 3.1 原始字段（直接行情）

- `symbol`：股票代码
- `name`：股票名称
- `theme`：所属行业（作为题材代理）
- `pct_chg`：涨跌幅
- `turnover`：换手率
- `amount`：成交额
- `volume_ratio`：量比
- `amplitude`：振幅

## 3.2 衍生字段（程序计算）

- `industry_mean_pct`：所在行业平均涨跌幅
- `industry_up_ratio`：所在行业上涨家数占比

## 3.3 策略评分字段（0-100）

- `theme_strength`：行业强度（行业平均涨幅 + 行业成交额）
- `sector_linkage`：板块联动（行业平均涨幅 + 上涨占比）
- `stock_strength`：个股强度（涨跌幅 + 量比 + 振幅）
- `capital_support`：资金承接（换手率 + 成交额 + 量比）
- `total_score`：四因子按权重加权总分

## 3.4 风险字段

- `risk_tag`：按振幅和换手率分为低/中/高波动

## 4. 如何更新数据

## 方式 A：UI 内一键刷新（推荐）

1. 打开页面
2. 点击“刷新实时数据”
3. 再点击“运行筛选”

## 方式 B：命令行自动化

```bash
python scripts/update_live_data.py
```

可配合 Windows 任务计划每天定时执行。
